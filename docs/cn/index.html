本地 AI 围棋复盘工作台

天书阁（TensuGo）：把 KataGo 的判断，放回你的棋盘和变化图里。

天书阁（TensuGo）是一款桌面围棋研究应用。它导入 SGF，连接本地 KataGo， 展示候选点、胜率、目差、访问量和 PV，并把分支研究整理成可阅读的复盘资料。

下载 Windows MSI 下载 macOS DMG GitHub

Windows x64 内置 KataGo OpenCL 引擎。macOS Apple Silicon 版本需要另行安装 KataGo。

为什么做它

不是直播评估器，而是复盘时真正会用的案头工具。

棋局研究经常在棋盘、AI 终端、SGF 编辑器和笔记之间来回切换。 天书阁（TensuGo）把这些动作收束到一个安静的桌面界面里： 你可以沿着棋谱树研究，预览候选变化，保留分支，并把关键图形和文字说明沉淀下来。

功能

围绕复盘，而不是围绕参数。

棋

SGF 与变化图

导入棋谱后保留分支结构，主分支、变化、删除和回到分歧点都按研究习惯组织。

AI

KataGo 本地分析

连接本机 KataGo，查看候选点、胜率、目差、访问量和 PV，不需要把棋谱上传到云端。

图

候选点与趋势

候选气泡、列表和胜率走势同步显示，适合快速定位胜负手、缓手和方向选择。

文

研究文档

把局面图、候选变化和文字解说放进研究文档，便于赛后总结、教学和资料归档。

工作流

从一盘棋到一份复盘。

  1. 导入

    打开 SGF，沿主分支或任意变化进入局面。

  2. 分析

    启动本地 KataGo，查看候选点和 AI 推荐变化。

  3. 研究

    保留有价值的分支，比较不同下法的目差和走势。

  4. 整理

    生成带局面图和说明的复盘资料。

下载

Windows 和 macOS 都可以开始复盘。

大多数用户会在 Windows 上使用天书阁（TensuGo），所以 Windows x64 版本已经发布， 并内置 KataGo OpenCL 引擎。macOS Apple Silicon 版本保持轻量，需要用户自行安装 KataGo。

Windows x64 (64-bit)

内置 KataGo OpenCL 引擎，2026.5 版本。

下载 Windows MSI
macOS Apple Silicon (M-series)

不内置引擎。请先通过 Homebrew 安装 KataGo： brew install katago

安装天书阁（TensuGo）后，打开 设置 > 引擎， 使用 Auto Detect 自动检测和配置引擎。也可以在同一面板手动下载并配置引擎。
